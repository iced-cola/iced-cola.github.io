设置git初始化默认分支为main

好久没上传代码到 GitHub 了，前两天新建了一个仓库，然后习惯性的执行 git push origin master  , 过了好一会儿才在 GitHub 页面上看到两个分支：main 和 master。

这两天陆陆续续出现好几次，网上查了一下才知道从10月份起，GitHub 的默认分支名不再是master，而是main。据说是因为master这个词和种族主义有关系，现在的人真是玻璃心啊。
